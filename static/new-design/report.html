CVE-2024-3094 · XZ Utils 后门

CVE-2024-3094 是位于 liblzma(由 xz-utils 提供)中的供应链植入, 可在特定构建配置下通过 OpenSSH(systemd-notify 链路)实现远程未授权代码执行。 受影响版本:5.6.0、5.6.1。

1. 概要

该后门由长期合规贡献者悄然植入。仅在 x86_64、glibc 系发行版构建中触发, 绕过条件依赖 RPM_ARCH 与 DEB_HOST_GNU_TYPE 等环境变量。

  • 影响入口:OpenSSH 通过 libsystemd 间接链接 liblzma。
  • 触发路径:RSA 公钥校验阶段,RSA_public_decrypt 被替换。
  • 检测难度:载荷以混淆形式藏于二进制测试文件,源码仓库无显式可疑代码。

2. ATT&CK 映射

主要落在 T1195.002 与 T1078 。在防御侧,触达 M1051 DS0022 。

3. 关键 IOC(节选)

首次落盘的恶意构建产物哈希 #a3f5e9c7…b21c ,经测试在 IP203.0.113.42 发出回连;域名 DNjiat75[.]example 被用作开发流水线下载源。完整列表见「IOC 列表」标签。

4. 缓解 / 处置建议

  1. 立即识别 xz-utils 5.6.0 / 5.6.1 装机情况;回滚到 5.4.x。
  2. 核对 liblzma 来源签名。
  3. 启用「Sigma 规则」标签生成的检测规则。
